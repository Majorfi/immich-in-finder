# Findich — landing site

Marketing site for [Findich](https://github.com/Majorfi/immich-in-finder) — an
Immich drive for the macOS Finder. In the spirit of getwhimbrel.app. Next.js 15
(App Router) + Tailwind CSS v4, static one-pager, deployed on Vercel (root
directory: `site/`).

```bash
npm install
npm run dev    # http://localhost:3000
npm run build
```

The Finder window in the hero is pure CSS/SVG — no screenshots to keep in sync.
